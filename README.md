# Arithmetic 

Arithmetic is a Sugar activity that helps learners improve their mental math skills through interactive, timed arithmetic puzzles.

# Features

1. Multiple difficulty levels (<10, <20, <50) to support progressive learning.

2. Supports basic arithmetic operations:

* Addition

* Subtraction

* Multiplication

* Division
* Factorial
* Modular Addition

3. Generates questions based on selected difficulty and operation.

4. Provides immediate feedback for each response.

5. Tracks scores per round and overall performance.

6. Displays response time for each question.

# How to Use

1. The player selects a difficulty level.

2. The game generates arithmetic questions based on the chosen difficulty.

3. The player answers each question within a time limit.

4. Immediate feedback is shown after each answer.

5. Scores and response times are recorded and displayed in a tabular format.

# Use Cases

Arithmetic is designed for:

* Practicing mental math skills.
  
* Self-learning and classroom activities.

* Learners of different age groups through adjustable difficulty.
